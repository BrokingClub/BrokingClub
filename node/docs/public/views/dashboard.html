Cucumber testing

The individual pages of Broking Club are being tested using the Cucumber testing framework. Features are written in plain text which is easy to understand for business people and executed using regular expressions and JavaScript.

All of our implemented acceptance tests can be found on this page, they will be executed in real-time as soon as you select a feature.
